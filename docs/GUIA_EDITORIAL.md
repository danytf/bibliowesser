# Guía Editorial --- Biblioteca de ONG

## Objetivo

Este documento define las normas editoriales que deben aplicarse a todas
las fichas de la biblioteca.

Su finalidad es **mejorar la legibilidad**, nunca modificar el
contenido.

La revisión editorial se realiza **después** de haber validado la
veracidad y completitud de la información.

------------------------------------------------------------------------

# Principio fundamental

**Editar la forma, no el fondo.**

Una revisión editorial no debe:

- añadir información;
- eliminar información relevante;
- modificar cifras;
- cambiar el significado;
- alterar el tono;
- resumir contenido;
- ampliar contenido.

Solo debe hacer que el mismo contenido sea más fácil y agradable de
leer.

------------------------------------------------------------------------

# Reglas editoriales

## 1. Un párrafo = una idea

Cuando cambie la idea principal, comienza un nuevo párrafo.

No crear bloques de texto excesivamente largos.

------------------------------------------------------------------------

## 2. Priorizar el ritmo visual

Evitar "paredes de texto".

Como referencia:

- Preferiblemente entre 2 y 4 frases por párrafo.
- Evitar párrafos muy largos salvo que sea imprescindible.

------------------------------------------------------------------------

## 3. Dividir frases excesivamente largas

Si una frase supera aproximadamente 35--40 palabras y puede dividirse
sin alterar el significado, hacerlo.

Nunca dividir una frase si empeora la comprensión.

------------------------------------------------------------------------

## 4. Mantener el orden lógico

No reordenar ideas.

No cambiar la estructura narrativa.

Solo mejorar la presentación.

------------------------------------------------------------------------

## 5. Mantener el estilo original

Conservar:

- vocabulario;
- tono;
- nivel técnico;
- ejemplos;
- cifras.

No "reescribir" por gusto.

------------------------------------------------------------------------

## 6. No simplificar contenido

La misión no es resumir.

Todo el contenido importante debe permanecer.

------------------------------------------------------------------------

## 7. Consistencia

Aplicar los mismos criterios a todas las ONG para que la experiencia de
lectura sea homogénea.

------------------------------------------------------------------------

# Criterios de aceptación

Una revisión editorial es correcta si:

- El lector entiende exactamente lo mismo que antes.
- No se ha perdido información.
- No se ha añadido información.
- La lectura resulta claramente más cómoda.
- Los párrafos descansan la vista y separan ideas.

------------------------------------------------------------------------

# Instrucciones para el asistente

Cuando se solicite aplicar esta guía:

1. Revisa únicamente la ficha indicada.
2. Aplica exclusivamente estas normas editoriales.
3. No modifiques el contenido.
4. Si un texto ya cumple la guía, déjalo intacto.
5. Devuelve únicamente el texto revisado.
6. No justifiques cambios salvo que se solicite expresamente.

------------------------------------------------------------------------

# Filosofía

El objetivo no es escribir diferente.

El objetivo es que un captador pueda leer una ficha completa con menos
esfuerzo y mayor comprensión, manteniendo intacto el contenido y la
intención original.
